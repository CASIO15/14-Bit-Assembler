◆ debug_list_destroy()

void debug_list_destroy ( debugList **  list)

Destroys a debugList and all its nodes.

Parameters
listA pointer to the debugList to be destroyed.
Note
After calling this function, the pointer to the debugList should be set to NULL. This function frees the memory allocated for a debugList and all its nodes. It first iterates over the linked list of debugNodes, and for each node, it calls debug_list_destroy_node to free any additional memory that was allocated for it, and then it frees the node itself. After all nodes are freed, it frees the memory allocated for the debugList itself.
See also

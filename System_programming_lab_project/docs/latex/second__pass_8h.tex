Generates an object file from the data in a memory buffer.

This function takes a {\ttfamily \mbox{\hyperlink{structmemory_buffer}{memory\+Buffer}}} structure and a file path, and generates an object file containing the data from the memory buffer. The object file is created using the {\ttfamily translate\+\_\+to\+\_\+machine\+\_\+data()} and {\ttfamily get\+\_\+outfile\+\_\+name()} functions, and it is written to the specified path.

\end{DoxyParams}
